世界観（文明崩壊後の未来）

文明崩壊から千年、地上に戻った人類はモンスターと共に暮らす。
このページは、歴史・景観・モンスター・暮らし・物語・プレイヤーの立場を整理し、ゲーム全体の一貫性を支える設定資料です。

目次

📖 記事概要 #

文明崩壊後の未来を舞台に、人類の地上回帰、モンスターの進化、生存のための暮らしや探索、そしてプレイヤーが体験するクエストの方向性をまとめています。ゲームデザインやストーリーの一貫性を保つための基盤となる内容です。

1. 世界の歴史 #

文明崩壊 #

  • かつて繁栄していた高度な文明は、突如として滅びた。
  • 理由は最後まで不明であり、伝承や憶測だけが残されている。
  • 人類は生き延びるために地下に避難し、1000年間を地下で暮らした。
  • 一部は混乱期に宇宙へ脱出したという噂も残る。
  • 1000年の間に、旧来の動物は姿を消し、神話を想起させる存在が現れる。

地上への帰還 #

  • 約100年前、人類は地上へ復帰。
  • 自然が廃墟を侵食した荒廃世界が広がっていた。
  • 各地に旧文明の遺跡が点在し、小さなコミュニティが生まれた。

2. 世界の景観 #

  • 廃墟と自然の融合：森に飲み込まれた都市跡、砂に埋もれた鉄道、崩れたダムや水没都市。
  • 遺跡：碑文や装置が断片的に作動。滅亡理由は依然として謎。

3. モンスター #

起源 #

  • 環境変化の中で動植物が進化・変異して誕生。
  • 神話的イメージを持ちながらも、新たな生態系の構成要素。
  • 人類にとって脅威であり、時にパートナー。

デザイン指針 #

  • 自然進化として説明できるフォルム。
  • 環境適応（炎＝火山地帯、水＝水没都市、風＝高層廃墟、土＝荒野…）。
  • 神話性と合理性の両立。

4. 人類の暮らし #

  • 地上復帰後は小規模コミュニティで生活。
  • 遺跡探索と資源採取で生計を立てる。
  • 探検者・研究者は旧文明の手掛かりを求めて危険地帯へ。

5. ストーリーとクエスト #

ストーリー #

  • 文明崩壊の理由は語られず、常に謎として漂う。
  • プレイヤーはモンスターと共に探索し、断片的な伝承や遺物に触れる。
  • 「宇宙へ旅立った人々」や「神話的存在」の記録が点在。

クエスト例 #

  • 遺跡探索：建造物内部での遭遇戦。
  • 資源収集：崩壊都市・自然地帯での採集。
  • 生活支援：コミュニティからの依頼を達成。
  • 特殊環境バトル：砂嵐の荒野、氷湖の決闘など。

6. プレイヤーの立場 #

  • 新時代を生きる若者。
  • モンスターを育成し、探索・対戦へ導く。
  • 目的は 育成・対戦・探索 に集約。

7. 現実世界との関係 #

  • 「人気者になること」は現実のプレイヤー体験の話。
  • プレイ／配信／発信によって現実のSNSで注目を得る設計。
  • 世界観とゲームシステムは没入体験を支え、現実との接点は外部に置く。

💡 まとめ #

このゲームは、文明崩壊後の自然と遺跡が混ざる世界を舞台に、プレイヤーがモンスターと共に生き抜き、探索・育成・対戦を通じて物語を紡ぐ。滅亡理由は最後まで不明で、宇宙へ脱出した人々の記録や神話的存在の出現が、謎とロマンを添える。
