# MyGame 公式デモサイト
GitHub Pages で公開するブラウザ向け対戦デモ。コンセプト/要件/データを併載。

## 公開URL
https://wakapai-game.github.io/mygame/

## 構成
- demo.html: デモ実行
- concept/: 全体/育成/対戦のコンセプト
- requirements/: 要件定義
- data/: monsters.json / moves.json / types.json / version.json
- pages/: JSONを表表示するビュー
