募集職種と内容

① AIエンジニア（活用推進・自動化）

AIを道具として最大活用し、開発効率を底上げする実務推進ポジション（プロンプト×自動化）。

  • AI活用基盤の整備（プロンプト設計・自動化・ナレッジ共有）
  • コーディング支援（HTML/CSS/JS/TypeScript など）
  • AgentBuilder／LangChain 等の調査・実装
  • AIによるドキュメント生成／リファクタリング

② コンセプトアート（世界観・UI/UX 提案）

提出アートは方向性検討の素材です。最終採用・デザイン決定は代表が行います（不採用の可能性あり）。

  • タイトル画面／対戦画面などの画面デザイン案
  • 背景・モンスター等のコンセプトアート
  • 「遊びたくなる」絵・ビジュアルの提案

③ ゲームプログラマー（対戦システム）

コマンドバトルの中核ロジック設計・実装を担当。

  • ブラウザ上で動作する対戦システムの設計・実装
  • 技・ステータス・属性相性・効果処理などのロジック
  • Canvas／WebGL を用いた演出・UI制御
  • GitHub でのコード管理・共同開発

参加方法・働き方

  • 完全リモート／自由参加型
  • 毎日5分でもOK — 小さなアウトプットを継続（例：コード1行、デザイン案1枚、AIプロンプト1個）
  • コミュニケーション：基本は Slack でチャット、週1回オンラインMTG（雑談・疑問共有・Q&A）
  • 年齢性別不問／無償（趣味の延長で参加できる方）
  • 最終意思決定権は代表に一任
  • 1ヶ月単位で参加継続の意思を相互に確認
  • 将来的にチームでSNS発信・YouTubeなども検討

応募条件

  • GitHub での開発・管理（Pull Request／Issue／Commit など）ができる
  • Slack でチャット連携できる（Discordでの音声MTGも可）
  • 指示を正確に理解し、柔軟に対応できる
  • チームの方向性を尊重し、協調的に進められる
